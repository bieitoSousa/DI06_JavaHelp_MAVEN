Fichero mapa de JavaHelp

Cabecera :

  Definimos un Fichero XML {<?xml version='1.0' encoding='ISO-8859-1'?>}

  Definimos tipo de Documento : map {<!DOCTYPE map>}

Contenido :

 Definimos un Espacio de nombres map {<map version="1.0">[...] </map> }

    Creamos una entrada {<mapID /> } por cada uno de los html que creamos :

      1. una "clave" {target="clavedelfichero"}

      2. una "ruta" {url="html/main.html}

Nota aclaratoria:

 Es decir, a cada fichero html se le da un nombre, que servirá para identificarlo a partir de ahora.

 Tanto el resto de los ficheros de configuración de JavaHelp como nuestro código Java referenciarán a los ficheros .html con el nombre que pongamos en este fichero mapa.

 El fichero mapa tiene extensión .jhm -Java Help Map-, y en nuestro ejemplo será map_file.jhm. Lo colocaremos debajo del directorio help, en parelelo con el subdirectorio html.

Codigo :

<?xml version='1.0' encoding='ISO-8859-1'?>
<!DOCTYPE map
PUBLIC "-//Sun Microsystems Inc.//DTD JavaHelp Map Version 2.0//EN"
"http://java.sun.com/products/javahelp/map_2_0.dtd">
<map version="1.0">
   <mapID target="definicion" url="html/JavaHelp_definicion.html" />
   <mapID target="map_file" url="html/map_file.html" />
   <mapID target="toc_file" url="html/toc_file.html" />
   <mapID target="indice_file" url="html/indice_file.html" />
   <mapID target="Carpeta_JavaHelpSearch" url="html/Carpeta_JavaHelpSearch.html" />
   <mapID target="help_set_file" url="html/help_set_file.html" />
</map>
