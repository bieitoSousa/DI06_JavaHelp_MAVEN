Fichero índice de JavaHelp

Cabecera :

  Definimos un Fichero XML {<?xml version='1.0' encoding='ISO-8859-1'?>}

  Definimos tipo de Documento : index {<!DOCTYPE map>}

Contenido :

 Definimos un Espacio de nombres index {<index version="1.0">[...] </index> }

    Dentro de este espacio aplicamos ordenamos las entradas segun tu criterio , metiendo unas entradas {<indexitem /> } antes que otras.

    Creamos una entrada {<indexitem /> } por cada uno de los html que creamos :

      1. mostramos , un "texto" {text="textoMostrado"}

      2. hacemos referencia , una "clave" {target="clavedelfichero"}

Nota aclaratoria:

  El fichero index_file tiene extensión .xml en nuestro ejemplo será index_file.xml. Lo colocaremos debajo del directorio help, en parelelo con el subdirectorio html.

Codigo :

<?xml version="1.0" encoding="ISO-8859-1"?>
<index version="1.0">
   <indexitem text="[entrada 0] JavaHelp" target="definicion">
      <indexitem text="[entrada 1] Mapa" target="map_file"/>
      <indexitem text="[entrada 2] Tabla de Contenidos" target="toc_file"/>
      <indexitem text="[entrada 3] Indice" target="indice_file"/>
      <indexitem text="[entrada 4] Pestaña Busqueda" target="Carpeta_JavaHelpSearch"/>
      <indexitem text="[entrada 5] Configuracion" target="help_set_file"/>
   </indexitem>
</index>
