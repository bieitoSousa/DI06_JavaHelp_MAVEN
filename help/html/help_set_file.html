Fichero configuración de JavaHelp

Este fichero lo copiamos de un template y editamos sus campos

Cabecera :

  Definimos un Fichero xml {<?xml version='1.0' encoding='ISO-8859-1'?>}

  Definimos tipo de Documento : helpset {<!DOCTYPE helpset>}

Contenido :

 Definimos un Espacio de nombres helpset {<helpset version="1.0">[...] </helpset> }

    Dentro de este espacio de nombres creamos entradas {<title />    <maps />    <view /> }

    Entrada Titulo {<title > nombreTitulo <title /> }

    Entrada MAPS {<maps > [...] <maps /> }

      Defimos Pagina por defecto de Ayuda : <homeID > targetdelFicheromap_file <homeID />

      Defimos el map :<mapref location = "map_file.jhm" />

    Entrada VIEW {<maps > [...] <maps /> } definimos 3 {Tabla Contenidos | Indice | Buscar}

      Defimos NOMBRE: <name > {Tabla Contenidos | Indice | Buscar} <name />

      Defimos ETIQUETA: <label > {Tabla de contenidos | El indice | Buscar} <label />

      Defimos TIPO: <type > {javax.help.TOCView | javax.help.IndexView | javax.help.SearchView} <type />

      Defimos DATOS: <data > {toc.xml | indice.xml | engine="com.sun.java.help.search.DefaultSearchEngine" > JavaHelpSearch } <data />

Nota aclaratoria:

  En --> view/name=Buscar --> data necesita el atributo engine ="" y pasarle la carpeta {JavaHelpSearch}

Codigo :

<?xml version="1.0" encoding='ISO-8859-1' ?>
<!DOCTYPE helpset
PUBLIC "-//Sun Microsystems Inc.//DTD JavaHelp HelpSet Version 1.0//EN"
"http://java.sun.com/products/javahelp/helpset_1_0.dtd">
<helpset version="1.0">
   <title>Ejemplo ayuda JavaHelp</title>
   <maps>
      <!-- Pagina por defecto al mostrar la ayuda -->
      <homeID>definicion</homeID>
      <!-- Que mapa deseamos -->
      <mapref location="map_file.jhm"/>
   </maps>

   <!-- Las Vistas que deseamos mostrar en la ayuda -->
   <!-- La tabla de contenidos -->
   <view>
      <name>Tabla Contenidos</name>
      <label>Tabla de contenidos</label>
      <type>javax.help.TOCView</type>
      <data>toc.xml</data>
   </view>

   <!-- El indice -->
   <view>
      <name>Indice</name>
      <label>El indice</label>
      <type>javax.help.IndexView</type>
      <data>indice.xml</data>
   </view>

   <!-- La pestana de busqueda -->
   <view>
      <name>Buscar</name>
      <label>Buscar</label>
      <type>javax.help.SearchView</type>
      <data engine="com.sun.java.help.search.DefaultSearchEngine">
         JavaHelpSearch
      </data>
   </view>

</helpset>
