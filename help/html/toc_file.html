Fichero de Tabla de Contenidos de JavaHelp

Esta tabla de contenidos no es más que una especie de "árbol".

Creando elementos principales y dentro de los mismos los seundarios y asi sucesibamente

Cabecera :

  Definimos un Fichero XML {<?xml version='1.0' encoding='ISO-8859-1'?>}

Contenido :

 Definimos un Espacio de nombres toc {<toc version="1.0">[...] </toc> }

    Dentro de este espacio aplicamos gerarquias , metiendo unas entradas {<tocitem /> } dentro de otras.

    Creamos una entrada {<tocitem /> } por cada uno de los html que creamos :

      1. mostramos , un "texto" {url="textoMostrado"}

      2. hacemos referencia , una "clave" {target="clavedelfichero"}

Nota aclaratoria:

 El fichero toc_file tiene extensión .xml en nuestro ejemplo será toc_file.xml. Lo colocaremos debajo del directorio help, en parelelo con el subdirectorio html.

Codigo :

<?xml version="1.0" encoding="ISO-8859-1"?>
<toc version="1.0">
   <text="[FICHERO] JavaHelp" target="definicion"/>
      <tocitem text="[FICHERO] Mapa" target="map_file"/>
      <tocitem text="[FICHERO] Tabla de Contenidos" target="toc_file"/>
      <tocitem text="[FICHERO] Indice" target="indice_file"/>
      <tocitem text="[FICHERO] Pestaña Busqueda" target="Carpeta_JavaHelpSearch"/>
      <tocitem text="[FICHERO] Configuracion" target="help_set_file"/>
   </tocitem>
</toc>
